# Portfolio

Personal portfolio site — live at [AdvaithK500.github.io](https://AdvaithK500.github.io).

Single-file HTML, zero dependencies, deploys instantly to GitHub Pages.

## Deploy

```bash
git clone https://github.com/AdvaithK500/AdvaithK500.github.io
# edit index.html — replace all AdvaithK500 / YOUR_EMAIL / YOUR_LINKEDIN placeholders
git add . && git commit -m "update portfolio" && git push
```

## Customise

All placeholders to replace in `index.html`:

| Placeholder | Replace with |
|---|---|
| `AdvaithK500` | Your GitHub username |
| `YOUR_EMAIL@e.ntu.edu.sg` | Your NTU email |
| `YOUR_LINKEDIN` | Your LinkedIn handle |
| `resume.pdf` | Add your resume PDF to this folder |
